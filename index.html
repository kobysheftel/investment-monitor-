    How to use
    1. Pick a base style (1–10) with the number keys below the table.
    2. Areas — tweak colors, font size, and bold for each zone (Title / Header / Body / Buttons). Changes layer on top of the chosen style.
    3. Directives — type a rule and hit Apply. Examples:
     · buttons: pill / square / rounded
     · theme: calmer / louder
     · header: subtle / bigger
     · body: compact / spacious
     · desc-bg: #1e293b — description panel color
    4. Export — saves the full config as JSON.
